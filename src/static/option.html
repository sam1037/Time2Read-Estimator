Blacklist Management

Configure your blacklist url patterns below:

URL Pattern Remove?
https://developer.chrome.com/docs/extensions/reference/manifest/default-locale

Use '*' to represent any subsequent part of the URL.

For example, the pattern "https://developer.chrome.com/docs/*" will disable the extension on the following websites:

1. https://developer.chrome.com/docs/extensions/reference/manifest/default-locale

2. https://developer.chrome.com/docs/webstore/prepare

3. https://developer.chrome.com/docs/web-platform/chrome-flags
